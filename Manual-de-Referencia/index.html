Introdução

O presente trabalho tem por finalidade desenvolver, de forma integral e estruturada, uma página em HTML que funcione como um manual de referência.

Para cada tag estudada, será apresentada sua definição, a descrição de sua finalidade e a exemplificação prática de sua utilização por meio de trechos de código funcional.

A página será organizada em seções distintas, cada uma dedicada a um conjunto específico de tags.

Utilizamos como referência essencialmente o site W3Schools.com.

Também utilizamos as bibliotecas prismjs.com, prism-markup e prism.css para melhorar a formatação e deixar o código mais legível. Para isso contamos com a boa e velha ajuda do Google para ajusta-lo no código.

Tags de Estrutura e Texto

<h1> até <h6>

As tags <h1> até <h6> são usadas para definir cabeçalhos HTML. <h1> define o cabeçalho mais importante, enquanto <h6> define o menos importante. Essas tags ajudam a organizar o conteúdo em seções e subseções, melhorando a legibilidade e a estrutura do documento. Abaixo o exemplo de uso no código HTML e o resultado visual.

Exemplo de código:


<h1>Exemplo de cabeçalho 1</h1>
<h2>Exemplo de cabeçalho 2</h2>
<h3>Exemplo de cabeçalho 3</h3>
<h4>Exemplo de cabeçalho 4</h4>
<h5>Exemplo de cabeçalho 5</h5>
<h6>Exemplo de cabeçalho 6</h6>

Tags Multimídia no HTML5

<video>

A tag <video> é usada para incorporar e mostrar vídeos em uma página HTML. Os atributos controls (exibe controles de reprodução), como play, pause e volume. O autoplay (inicia automaticamente), loop (reproduz em repetição) e muted (silenciado por padrão). É também altamente recomendado incluir width e height para definir o tamanho do vídeo e evitar problemas na página. O elemento <source> dentro da tag <video> permitem videos alternativos com diferentes formatos que o navegador pode escolher. O navegador vai usar o primeiro formato que ele suportar. Importante ainda lembrar que o texto entre a tag de abertura e fechamento do <video> é exibido caso o navegador não suporte o elemento de video.

Exemplo de código:


<video width="320" height="240" controls>
  <source src="video-exemplo.mp4" type="video/mp4">
  Seu navegador não suporta a tag de vídeo.
</video>

Resultado visual:

Video exemplificativo retirado de W3School.com. Cortesia de Big Buck Bunny.

<audio>

A tag <audio> é usada para inserir arquivos de áudio em uma página HTML. Tal como no vídeo, pode-se usar atributos como controls, autoplay e loop. O atributo controls é imprescindível para que o usuário reproduza (play) ou pause o áudio.

Exemplo de código:


<audio controls>
  <source src="audio-exemplo.mp3" type="audio/mpeg">
  Seu navegador não suporta a tag de áudio.
</audio>

Tags de Formulários no HTML5

<form>

A tag <form> é usada para criar formulários. Ela contém campos de entrada, botões, menus suspensos e outros elementos que permitem ao usuário enviar dados o informações. Os atributos mais comuns são action (define para onde enviar os dados) e method (GET ou POST).

Exemplo de código:


<form action="/enviar" method="post">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome">
  <button type="submit">Enviar</button>
</form>

<input>

A tag <input> cria campos interativos em formulários. É provavelmente o elemento mais utilizado em formulários O atributo type define o tipo de dado aceito, tal como texto, número, email, senha, checkbox, radio etc.

Exemplo de código:


<input type="text" placeholder="Digite seu nome"><br>
<input type="email" placeholder="Digite seu e-mail"><br>
<input type="password" placeholder="Digite sua senha"><br>
<input type="checkbox" id="aceito">
<label for="aceito">Aceito os termos</label>

<summary>

A tag <summary> define um título visível para o elemento <details>. Ao clicar no título, o conteúdo adicional pode ser exibido ou ocultado. É útil para FAQs, descrições ou informações adicionais que não precisam estar sempre visíveis.

Exemplo de código:

</details>

<select>

A tag <select> cria uma lista suspensa (dropdown) em formulários. Cada opção é definida dentro de um elemento <option>. O atributo name identifica o campo, e o atributo value em <option> define o valor enviado no formulário.

Exemplo de código:


<form>
  <label for="linguagem">Escolha uma linguagem:</label>
  <select id="linguagem" name="linguagem">
    <option value="html">HTML</option>
    <option value="css">CSS</option>
    <option value="javascript">JavaScript</option>
    <option value="python">Python</option>
  </select>
</form>

<script>

A tag <script> é usada para inserir ou fazer referencia a códigos em JavaScript dentro da página HTML. Pode ser usada com código escrito diretamente no documento ou importando de arquivos externos via atributo src. É comumente usado para validação de formulários, manipulação de imagens e outras modificações dinâmicas.

Exemplo de código:


<!-- Script interno -->
<script>
  function mostrarMensagem() {
    alert("Olá Mundo!!");
  }
</script>

<button onclick="mostrarMensagem()">Clique aqui</button>

<!-- Script externo -->
<script src="meu-script.js"></script>

Resultado visual:

O botão executa um código JavaScript ao ser clicado. Também é possível carregar um arquivo externo, tal como mostrado na última linha do código.
